To use or see the project you must have XAMPP/MAMPP/WAMPP installed in your pc.
setup: 
	1. goto xampp/htdocs if you are using xampp and paste the whole file in a directory.
	2. Create a database in phpmyadmin named "elibrary" or change the $dbName into "_include/db/Database.php"
	3. open "_include/db/Database.php" and set the hostName,passdb,userdb accourding to your database setup in your phpmyadmin
	4. I have provided a db backup in "_include/table backup" so import the database in your database called "elibrary" or the name 	   you provided
	5. visit localhost/{your-folder-name} and start using :)
